# Astromouse

This is a small game made in Godot. It is available at https://sourabhvyas.github.io/astromouse/

## Controls

- Keyboard: use Left and Right arrow keys to move between lanes.
- Touchscreen: tap the left or right side of the screen to move lanes.

## Gameplay

Gameplay is similar to Lumberjack.
